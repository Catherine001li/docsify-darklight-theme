## What it is

**docsify-darklight-theme** provides `dark-mode` and `light-mode` with switcher, so you can switch between these modes

## Features

- Theme switcher
- Theme customizable using docsify config object
- Config Default theme
- Theme rememberer
- Redesigned searchbox
